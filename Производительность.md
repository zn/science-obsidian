# Производительность

Тип: Понятие
Related: [[Архитектура ПО]] [[Нефункциональные требования]] [[Нагрузочная модель]] [[Перцентиль]]
Tags: #понятие #performance

Производительность - способность системы обрабатывать операции с заданными ограничениями по времени, пропускной способности, ресурсам и нагрузке.

## Исследовательский интерес

Производительность важна для [[Теория лимитов]], потому что выражает границы системы в измеримой форме: сколько запросов, за какое время, при какой нагрузке и на каких ресурсах система способна обработать.
