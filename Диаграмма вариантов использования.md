# Диаграмма вариантов использования

Тип: Понятие
Related: [[Архитектурная документация]] [[Принятие решений]] [[Онтология победы]]
Tags: #понятие #диаграммы #UML

Диаграмма вариантов использования показывает actors, system boundary и use cases, то есть высокоуровневые способы взаимодействия пользователей с системой.

## Исследовательский интерес

Use Case diagram связывает техническую систему с целями акторов. Для [[Онтология победы]] это полезно: "победа" системы может быть описана через успешное выполнение значимых use cases.
