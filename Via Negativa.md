# Via Negativa

Via negativa - способ мышления через отрицание, исключение и вычитание: приближение к хорошему решению через удаление плохого, вредного, лишнего или хрупкого.

## Исследовательский интерес

Для проекта via negativa важна как метод работы с ограничениями. Если невозможно точно указать лучший путь, можно начать с надежного исключения провальных путей, опасных состояний и хрупких зависимостей.
