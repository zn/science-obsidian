# Embedding

Тип: Понятие
Related: [[RAG]] [[Векторная база данных]] [[Искусственный интеллект]]
Tags: #понятие #ИИ #RAG

Embedding - векторное представление текста, кода или другого объекта в пространстве признаков. В RAG embeddings используются для поиска близких по смыслу фрагментов корпуса.

## Исследовательский интерес

Embedding-модель задает пространство сравнения. Смена модели меняет саму геометрию поиска, поэтому старые и новые embeddings нельзя бездумно смешивать.
